## Finding 1 (S1) — Data format mismatch breaks grading for ~13,000 questions

### Root cause

`src/components/QuizEngine.tsx` is letter-coded:

```tsx
// Line 26–27
const optionLabels = ["A", "B", "C", "D"];
const correctIdx = optionLabels.indexOf(q?.answer);   // expects "A"/"B"/"C"/"D"
// Line 40
if (selected === q.answer) { setScore((s) => s + 1); }  // selected is a letter
```

`selected` is always one of `"A"|"B"|"C"|"D"` (set by `onClick={() => handleSelect(label)}` where `label = optionLabels[idx]`). For grading to work, `q.answer` must also be a letter. No adapter layer exists in any `src/lib/*-quiz-data.ts` file — the raw JSON flows directly to the engine.

### Scope (measured)

| File | Total Qs | Letter-format | String-format | Broken? |
|---|---:|---:|---:|:---:|
| physics-quizzes.json | 2,600 | 0 | 2,600 | ❌ ALL broken (g11+g12) |
| chemistry-quizzes.json | 2,600 | 0 | 2,600 | ❌ ALL broken (g11+g12) |
| biology-quizzes.json | 2,600 | 0 | 2,600 | ❌ ALL broken (g11+g12) |
| maths-higher-quizzes.json | 2,600 | 8 | 2,592 | ❌ 99.7% broken |
| maths-quizzes.json | 10,400 | 7,810 | 2,590 | ❌ g11+g12 broken (1,295 each) |
| quizzes.json (English) | 10,396 | 7,796 | 2,600 | ❌ g11+g12 broken (1,300 each) |
| science-quizzes.json | 7,800 | 7,800 | 0 | ✅ fine (g5–g10) |
| cefr-*-quizzes.json (×4) | 7,150 | 7,150 | 0 | ✅ format fine (but see Finding 3) |

**Total broken: ~13,072 questions**, all in Grades 11–12 across six subjects.

### Live-site confirmation

- **URL:** `https://www.thepractiseground.in/quiz/physics/11/1?_=cb2026041401`
- **Question:** "Which of the following is NOT a fundamental force?" Options: A Gravitational / B Electromagnetic / C Strong nuclear / D Frictional
- **JSON stored answer:** `"Frictional"` (a full option string, not a letter)
- **User action:** Clicked option D (the factually correct answer)
- **Engine response:** Score = 0 correct so far, user's selection shown with a RED border, no option shown as green-correct
- **Conclusion:** Students answering grade 11/12 physics/chemistry/biology/maths-higher quizzes get 0/25 no matter what they pick.

### Why this happened

Prior placeholder-replacement batches on grade 11/12 subjects wrote each replacement question with `answer` set to the full option string (matching the `options` array). The earlier, hand-authored grade 5–10 content used letters. The regression is isolated to the replaced regions.

### Remedy options

1. **Rewrite the data (recommended):** Run a script that, for each question where `answer ∈ options`, replace `answer` with the corresponding letter (`"A"`–`"D"`). Fully reversible, preserves all content. Scope: ~13,072 questions.
2. **Patch the engine:** Teach QuizEngine to accept either a letter or a full option string. Lower risk of touching content but introduces runtime branching and doesn't fix the display text (`{answer}) {options[correctIndex]}` would render as e.g. "Frictional) undefined").

**Recommendation: Option 1** — cleaner, students see correct "A)/B)/C)/D)" labels in review.

## Finding 2 (S1) — Biology grade 11 has 44 weeks of the same question ×25

For `biology-quizzes.json`, grade `"11"`, weeks 9 through 52 (44 weeks), every week's 25 questions are **the same question copy-pasted 25 times**.

Examples:
- Week 9 — all 25 questions are `"Gymnosperms are:"`
- Week 10 — all 25 are `"The basis of animal classification primarily includes:"`
- Week 32 — all 25 are `"In the context of Biomolecules – Lipids: DNA contains:"` (note the out-of-place topic-prefix scaffolding)
- Week 52 — all 25 are `"The pituitary gland is controlled by:"`

This looks like an abandoned auto-generation scaffold that shipped without backfilling. Weeks 1–8 appear to have distinct questions; need confirmation.

**Remedy:** Regenerate grade 11 biology weeks 9–52 with 25 distinct NCERT-aligned MCQs per week (covering the named topic). Scope: 44 × 25 = 1,100 questions to author/curate.

## Finding 3 (S1) — CEFR answer letters are factually wrong in sampled cases

Format is letter-based (correct for the engine), but sample from `cefr-french-quizzes.json` a1, week 1 (Greetings & Introductions):

| # | Question | Options | Stored answer | Resolves to | Correct? |
|---|---|---|---|---|:---:|
| 1 | "What is the French greeting for 'Hello'?" | Bonjour / Bonsoir / À bientôt / Au revoir | `"D"` | **Au revoir** (Goodbye) | ❌ should be A (Bonjour) |
| 2 | "How do you say 'Good evening' in French?" | Bon matin / Bonjour / Bonne nuit / Bonsoir | `"C"` | **Bonne nuit** (Good night) | ❌ should be D (Bonsoir) |
| 3 | "What is the French word for 'Goodbye'?" | À plus tard / Bonjour / Au revoir / Bonsoir | `"C"` | **Au revoir** | ✅ |

2 of 3 in a single 3-question sample are wrong. Rate unknown until full audit. Suspected cause: answer letters were generated by a process disconnected from the actual options order. All 4 CEFR files (French/German/Spanish/English A2-B1) need a full re-validation pass.

**Remedy:** Fact-check all 7,150 CEFR questions. Can be parallelized across subagents.
